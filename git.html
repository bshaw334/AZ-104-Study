Learning Git

git init

Git init intializes git with the current directory to track changes.

git status

git status provides the current git status for the current directory.

git add

git add adds the current file or directory to the git queue for tracking.

git commit -m

git commit -m adds the changes to git with a message.

git log

git log provides a log of git inputs

git checkout *commit hash*

git checkout allows to restoring to previous commits based off of the commit hash under git status.

git branch

Branches allow to make an extension from the master branch without altering the master. Once you are ready you can merge the new branch with the master.

git branch

git branch displays the master/mian branch.

git branch *insert name*

git branch with name will create a branch from the master.

git merge *inesrt branch name*

git merge merges branches to the master.
