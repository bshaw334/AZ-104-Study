Creating Variables

Variables will make scripting simplified by taking known information and simplifying how it is presented to the script

Examples

Below I created a variable named $rg to pull in the name of a single resource group. This step can also be performed by typing/pasting a static resource group name.

rg$ = (Get-AzResourceGroup).ResourceGroupName

Below I created a variable named $RG to query the resource group list and list them in tab seperated value (tsv).

RG$ = az group list --query [].name -o tsv
